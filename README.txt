Συναρτήσεις που χρησιμοποιώ:

int parse(char *line, char **argv): τη χρησιμοποιώ αφότου διαβαστεί μια εντολή στο terminal ή από το batch file 
				    προκειμένου να διακρίνω τον αριθμό των εντολών που μου δόθηκαν (ανάλογα με 
				    τον αριθμό των ερωτηματικών -;- )

void Command(int i, char **argv): καλείται απο τη συνάρτηση execute, κατά τη δημιουργία των child processes, προκειμένου
				  να αποθηκέυσει πλεον την κάθε εντολή που έχει δωθεί στην εκάστοτε line στο αντίστοιχο
				  child process

int execute(char **argv, int children): τη χρησιμοποιώ για να εκτελέσω τις εντολές που δίνονται στο command line. Εδώ δημιουργώ
					τα child processes, ελέγχω να γίνεται σωστή διαχείρηση των κενών εντολών και της quit,
					εκτελώ τις εντολές και περιμένω μέχρι αυτές να τελειώσουν για να προχωρήσω μετά στην 
					επόμενη γραμμή

int ExitCheck(char *line): τη χρησιμοποιώ για να ελέγξω αν στην εκάστοτε γραμμή έχει δωθεί εντολή τερματισμού. Λειτουργέι σαν boolean.

Οι συναρτήσεις InteractiveMode και Batchmode είναι για τους 2 διαφορετικούς τρόπους που θέλω να λειτουργεί το shell μου. Το interactive 
διαχειρίζεται απλά line inputs από το χρήστη, ενώ το batch mode δέχεται ένα αρχείο και το επεξεργάζεται γραμμή προς γραμμή.

Η συνάρτηση main, ανάλογα με το όρισμα που της δίνεται, επιλέγει αν θα χρησιμοποιήσει την InteractiveMode ή τη BatchMode.
